HashTables

A Hash Table is a data structure that stores values associated with a key (key-value pairs). The Hash Table stores these values in an efficient manner by using a hash function. This function takes in the key and ideally returns a unique index number for the value to be stored in.

Delete Θ(1)
Search Θ(1)
