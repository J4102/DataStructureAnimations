Queue

SEE ANIMATION
Overview

A queue is a type of data structure where both ends of the data structure can be modified by following FIFO, first in first out. This means the element that is first inserted will be the first to be removed. By convention, the front of the and back of the queue are known as the head and tail respectively, following FIFO, first in first out.

Complexity
Function Average Case Worst Case
Insert Θ(1) O(1)
Delete Θ(1) O(1)
Search Θ(n) O(1)
Terms to Know
Queue Terms Definition
Enqueue insert an element at the tail of the queue
Dequeue remove an element at the head of the queue
