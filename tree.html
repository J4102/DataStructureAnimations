Tree

Definition

A tree is a data structure that has a hierarchical structure with root (a node with no parent) and subtrees of children (nodes with a parent node). A Binary Tree is a tree where all nodes have at most 2 children. What makes it a binary search tree is when the left child contains only nodes with values less than the parent node, whereas the right child only contains nodes with values greater than the parent node. Also there are no duplicates allowed.

Time Complexity

Insert - Θ(log(n))

Delete - Θ(log(n))

Search - Θ(log(n))

Function Time Complexity
Insert Θ(log(n))
Delete Θ(log(n))
Search Θ(log(n))
Tree Terms Definition
Root a node with no parent; there can only be one root
Leaf a node with no children
Siblings two nodes with the same parent
Height of a node length of the longest path from that node to a leaf (all leaves have height of zero)
Height of a tree maximum depth of a node in that tree = height of the root
Depth of a node length of the path from the root to that node
