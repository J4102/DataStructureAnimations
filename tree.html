A tree is a data structure that has a hierarchical structure with root (a node with no parent) and subtrees of children (nodes with a parent node). A Binary Tree is a tree where all nodes have at most 2 children. What makes it a binary search tree is when the left child contains only nodes with values less than the parent node, whereas the right child only contains nodes with values greater than the parent node. Also there are no duplicates allowed.
