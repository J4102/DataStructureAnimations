Definition

A Stack is a data structure that can only be changed from one side, the top of the stack. It has two main operations: push (insert) and pop (remove). Given the data structure's name, it follows LIFO (last in, first out), meaning the push method will add the element to the top of the stack, while the pop method will also remove the top element.
