Overview

A Stack is a data structure that can only be changed from one side, the top of the stack. It has two main operations: push (insert) and pop (remove). Given the data structure's name, it follows LIFO (last in, first out), meaning the element on top of the stack (which is also the last element to be pushed) would be popped out first.
