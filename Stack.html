Stack

A Stack is is used to store a collection of elements with two principal operations: push (insert) and pop (remove). Given the data structure's name, it follows LIFO (last in, first out), meaning the push method will add the element to the top of the stack, while the pop method will also remove the top element.

Stack Terms Definition
Push add an element at the top of the stack
Pop remove an element from the top of the stack
Top return the element at the top of the stack
