Overview

A linked list is a linear collection of elements/nodes in a sequence. The elements point to the next instead of having a physical placement in memory.

Singly-Linked List

A singly-linked list is traversed in one direction (forwards) which is from the head to the tail.

Doubly-Linked List

A doubly-linked list can be traversed in two directions (forwards and backwards). It does so by having a pointer that points to the next and previous nodes.

Circular Linked List

A Circular linked list can be traversed in one direction (fowards), similar to a singly-linked list or two directions (fowards and backwards), similar to a doubly-linked list. However, the last node which is the tail, points to the first node which is the head. Therefore, no node points to NULL.
